# Loan Approval Prediction

A machine learning project that predicts whether a loan application will be approved or rejected based on applicant details.

## Problem Statement
Automating the loan eligibility process based on customer details provided while filling an online application form such as Gender, Marital Status, Education, Income, Loan Amount, Credit History, and others.

## Dataset
- Training set: 614 records
- Test set: 367 records
- Source: Analytics Vidhya Loan Prediction Challenge

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- XGBoost

## Models Used
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 78.47% |
| Random Forest | 77.78% |
| XGBoost | 77.78% |
| Decision Tree | 64.58% |

## Project Flow
1. Data Collection
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing
4. Feature Engineering
5. Model Training & Evaluation
6. Prediction

## Conclusion
Logistic Regression achieved the best accuracy of 78.47% for this dataset.
